About Evaluation Metrics
Available Metrics

The ATLAS platform supports the following evaluation metrics:

Exact Match
Measures the percentage of tokens that match exactly between model output and ground truth.
Semantic Similarity
Measures the semantic closeness of model output to ground truth using embedding-based similarity.
Keyword Match
Calculates what percentage of key terms from the ground truth appear in the model output.
BLEU Score
An n-gram precision-based metric commonly used in machine translation evaluation.
ROUGE Score
Recall-oriented metric focused on n-gram overlap, useful for summarization tasks.
Metric Weights

Weights determine how much each metric contributes to the overall score. The weights should sum to 1.0 for proper normalization.

Target Threshold

The target threshold sets the success criteria for optimization. The workflow will attempt to achieve a score at or above this threshold.

Recommended Configurations
NLP Task-Specific Recommendations

  • Semantic Similarity (0.7): Prioritize meaning over exact wording
  • Keyword Match (0.2): Ensure key facts are captured
  • Exact Match (0.1): Minor consideration for precise answers
  • Target Threshold: 0.7

  • ROUGE Score (0.4): Evaluate recall of key information
  • Semantic Similarity (0.4): Ensure meaning is preserved
  • Keyword Match (0.2): Capture important terms
  • Target Threshold: 0.6

  • Exact Match (0.6): Precise syntax is important
  • Semantic Similarity (0.4): Capture intent and logic
  • Target Threshold: 0.8

  • BLEU Score (0.5): Standard translation metric
  • Semantic Similarity (0.5): Preserve meaning across languages
  • Target Threshold: 0.65
Usage Tips
  • Use weighted combinations of metrics for more robust evaluation
  • Adjust thresholds based on task difficulty - easier tasks can use higher thresholds
  • Consider the domain requirements when selecting metrics
